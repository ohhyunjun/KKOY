# 미세먼지 차단 자동 블라인드

> **홍익대학교 창의적공학설계(입문) - B조 KKOY 팀 프로젝트**
> 외부 대기질(미세먼지)을 실시간으로 감지하여 실내 유입을 자동으로 차단하는 스마트 블라인드 시스템입니다.

---

## 1. 프로젝트 개요

### 1.1 배경 및 필요성
* **미세먼지 문제 심화**: 최근 국내 전역에서 발생하는 고농도 미세먼지로 인해 호흡기 및 질환 발생 위험이 크게 증가하고 있습니다.
* **기존 환기의 한계**: 주기적인 환기는 실내 공기질 관리에 필수적이지만, 대기질이 나쁜 상태에서 과도한 환기는 오히려 외부 미세먼지를 실내로 대량 유입시키는 원인이 됩니다.
* **해결책**: 실시간 대기질을 모니터링하고 농도에 맞춰 스스로 개폐되는 **자동 블라인드 시스템**을 개발하여 안전하고 쾌적한 실내 환경을 유지하고자 합니다.

### 1.2 최종 아이디어 및 주요 기능
* **미세먼지 차단 특화 소재**: 일반 커튼 대신 먼지 차단 성능이 검증된 **미세먼지 방충망(차단 필터)**과 천을 결합하여 차단 효과를 극대화했습니다.
* **실시간 모니터링**: 미세먼지 센서 및 온·습도 센서를 탑재하여 대기 상태 데이터를 실시간으로 측정합니다.
* **데이터 시각화**: 수집된 수치를 LCD 판에 출력하고, 직관적인 3색 LED 모듈(좋음: 파랑 / 보통: 초록 / 나쁨: 빨강)을 통해 현재 상태를 쉽게 판별할 수 있습니다.
* **블라인드 제어 자동화**: 미세먼지 농도가 특정 수치($100\mu g/m^3$) 이상으로 나빠지면 블라인드가 자동으로 내려오고, 대기질이 개선되면 다시 자동으로 말려 올라갑니다.

## 4. 핵심 이슈 및 문제 해결 (Troubleshooting)

### Issue 1: 미세먼지 측정값 변동에 따른 모터의 과도한 떨림/반복 구동
* **문제점**: 미세먼지 센서 특성상 측정 수치가 실시간으로 미세하게 튀는 현상이 발생하여, 기준점 경계에서 모터가 열림과 닫힘을 계속해서 반복하는 현상이 일어남.
* **해결방안**: 수치가 일시적으로 변하더라도 모터가 급격히 반응하지 않도록 **타임 딜레이 알고리즘**을 추가함. 미세먼지 농도가 100 이상으로 감지되어 모터가 회전한 후, **최소 30초간 상태를 유지**하게 하고 그 이후에 대기질이 100 이하로 떨어졌을 때만 0도로 복귀하도록 소프트웨어 필터링을 적용함.

### Issue 2: 블라인드가 온전히 감기지 않는 서보모터의 회전 각도 제한
* **문제점**: 선정했던 `MG996R 서보모터`가 기본적으로 170도까지만 회전이 가능한 모델이어서 블라인드 천을 끝까지 감아올리거나 내리는 물리적 구동 범위가 나오지 않음.
* **시도 및 실패**:
  1. *스텝모터 교체*: 단독 구동은 잘 되나 기존 미세먼지/온습도 센서 라이브러리와의 타이밍 호환 문제로 통합 제어 실패.
  2. *DC모터 교체*: 센서 호환은 잘 되나 속도가 너무 빠르고 아두이노를 통한 정밀한 각도/위치 제어가 불가능하여 적합하지 않음.
* **최종 해결**: 기존 `MG996R 서보모터`를 분해하여 170도 제한을 거는 내부 물리적 **고정 핀을 제거**함. 이후 서보모터 내 내부 가변저항을 중앙에 맞춘 후 고정시켜 **360도 무한 회전(Continuous Rotation)이 가능하도록 물리 개조**하여 블라인드 상하강 롤링 문제를 완벽하게 해결함.
